The challenge

Moving research data shouldn't be hard

Microscopy instruments generate large datasets that need to reach project storage reliably and automatically.

Common pain points:

  • Manual copy is error-prone and slow
  • Transfers fail silently mid-way
  • No resume after interruption
  • Users forget to transfer new files
  • No visibility into transfer progress

Scion solves this with a desktop app that:

  • Monitors a local folder for changes
  • Transfers automatically via rsync
  • Resumes interrupted transfers
  • Verifies data integrity (checksums)
  • Notifies via real-time console output

How it works

Transfer Workflow

Choose DestBrowse remote folders
→
Select SourceLocal monitoring folder
→
Auto-syncrsync transfers data

Under the hood:

  • Authentication: LDAP login → SSH tunnel → rsync daemon started on server
  • File watching: Chokidar monitors the source folder for adds, changes, and new subdirectories
  • Transfer: rsync with --partial for resume, --append-verify for incremental large files
  • Transaction log: SQLite database tracks each transfer (start, resume, complete, cancel) per server
  • Stop: Process group kill (-pid) reliably terminates rsync and all children

Technical overview

Architecture (v1.2.12)

Vue 3 Composition API with TypeScript. LandingPage.vue orchestrates 11 composables:

ComposableRole
useAuthLDAP login, connection state
useRsyncProcess state, stop transfer
useDatabaseSQLite transaction tracking
useFileWatcherChokidar file monitoring
useTransferTransfer orchestration
useLayoutGoldenLayout + xterm
useThemeDark/light/system toggle
useAppStateElectron-store persistence

+ useRemoteFolders, useServer, useNetworkDrive

Tech Stack

  • Frontend: Vue 3, TypeScript, GoldenLayout
  • Desktop: Electron, @electron/remote
  • Transfer: rsync (via child_process)
  • Auth: ldapjs (LDAP), ssh2 (SSH)
  • Storage: SQLite (transactions), electron-store (prefs)
  • Terminal: @xterm/xterm
  • Styling: CSS custom properties (38 tokens)
  • Testing: Vitest (17 tests)
  • Build: Webpack + electron-builder
